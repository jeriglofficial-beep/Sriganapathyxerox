📧 EmailJS Setup — Used to send order responses to customers

How to set up (free):
1. Go to emailjs.com → Sign up free
2. Email Services → Add Service → Connect your Gmail (or any email)
3. Email Templates → Create Template → Use variables: {{to_email}}, {{customer_name}}, {{order_id}}, {{message}}, {{shop_name}}, {{total}}
4. Copy your Public Key (Account → API Keys), Service ID, and Template ID below

☁️ Cloudinary Storage — Unlimited file storage for customer uploads

How to set up (free — 25GB):
1. Go to cloudinary.com → Sign up free
2. From your Dashboard, copy your Cloud Name
3. Go to Settings → Upload → scroll to Upload Presets → click Add upload preset
4. Set Signing Mode to Unsigned → Save → copy the preset name
5. Paste both below and click Save
